Features

Powerful Features for
Secure Communication

Everything you need for private, offline messaging — packed into a lightweight PWA experience.

Bluetooth Messaging

Connect directly with nearby devices using advanced Bluetooth 5.0 technology. No WiFi, no cellular — just pure peer-to-peer communication.

Core Technology

E2E Encryption

256-bit AES encryption with secure key exchange protocols. Every message, file, and media is encrypted before it ever leaves your device.

Security

Group Chats

Create encrypted group conversations with multiple nearby devices. Perfect for team meetings, events, or any scenario requiring private group communication.

Collaboration

Media Sharing

Share images, videos, audio files, and documents seamlessly over Bluetooth. Fast transfer speeds with encrypted file delivery.

Media

100% PWA

Install directly from your browser — no app store needed. Works offline, loads instantly, and feels like a native app on any device.

Technology

Local Storage Only

All data stays on your device. Encrypted local storage ensures your messages are never uploaded to any server or cloud service.

Privacy
Security

Uncompromising
Security Architecture

Built from the ground up with privacy-first design. No compromises, no backdoors, no exceptions.

Secure Key Exchange

Diffie-Hellman key exchange protocol ensures that encryption keys are generated and shared securely between devices without any third-party involvement.

Device Authentication

Biometric and PIN-based device verification ensures only authorized users can access conversations and encrypted data stored locally.

Zero Knowledge Architecture

We can't read your messages — because we never receive them. Everything stays between your device and the connected peer. No servers, no logs, no tracking.

Self-Destructing Messages

Set timers on sensitive messages. Once the timer expires, the message is permanently erased from all connected devices — no traces left behind.

How It Works

Get Started in
Three Simple Steps

No sign-ups, no phone numbers, no email verification. Just install and connect.

01

Open the Web App

Visit Optibit in your browser. Tap "Add to Home Screen" from your browser menu to install it locally — no app store, no account, no tracking.

02

Enable Bluetooth

Turn on Bluetooth and scan for nearby Optibit users. The app automatically discovers compatible devices and initiates a secure handshake.

03

Start Chatting

Once connected, send encrypted messages, share media, and create group chats — all through Bluetooth with zero internet dependency.
